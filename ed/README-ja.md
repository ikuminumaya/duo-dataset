# EmpatheticDialogues 設定

## 概要
EmpatheticDialogues 設定では，ユーザは選択した感情ラベルに基づき，自身の個人的な経験を短く共有し，対話システムはユーザの感情的な発話に対して，適切に共感することを目的とする。
本設定には，ユーザ本人による主観評価と，第三者による客観評価の両方が含まれています。
